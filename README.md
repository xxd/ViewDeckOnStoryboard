##Intro
This is a simple example to show how to use Storyboard with [ViewDeck](https://github.com/Inferis/ViewDeck) .

##Compiling
Please follow the steps in [CocoaPods](https://github.com/CocoaPods/CocoaPods) to compile ViewDeck. 

Have fun!
